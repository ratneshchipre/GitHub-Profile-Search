# GitHub Profile Search

A dynamic web application to search and display GitHub user profiles with real-time data including followers, repositories, and bio information.

## Features
- Real-time GitHub profile search by username
- Display user avatar, name, and bio
- Show follower and following counts
- List public repository count
- Display user location
- Show top 4 recent repositories with direct links
- Clean and responsive user interface
- Keyboard support (Enter key to search)
- Input validation for empty searches
- Error handling for invalid usernames

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6 Async/Await)
- GitHub REST API
- Font Awesome Icons

## Installation
1. Clone the repository
```bash
   git clone https://github.com/ratneshchipre/GitHub-Profile-Search.git
```
2. Navigate to the project directory
```bash
   cd GitHub-Profile-Search
```
3. Open `index.html` in your browser
